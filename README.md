# ParthTraderAlerts Auto Pivot Telegram Bot

## Features

✅ Daily 09:15 AM IST Pivot Update  
✅ NIFTY 50 Pivot  
✅ BANKNIFTY Pivot  
✅ SENSEX Pivot  
✅ R1 R2 R3  
✅ S1 S2 S3  
✅ CPR Levels  
✅ Bullish/Bearish Bias  
✅ Telegram Automation


## Installation

Install requirements:

pip install -r requirements.txt


## Configuration

Update config.py:

BOT_TOKEN = "YOUR_TELEGRAM_TOKEN"

CHAT_ID = "YOUR_CHANNEL_ID"


## Run Bot

python bot.py


## Commands

/start
Start Bot

/pivot
Get Manual Pivot Levels

/help
Show Commands
